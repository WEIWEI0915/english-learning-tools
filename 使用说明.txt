本文件夹包含：
  index.html            — 互动学习网页（核心文件）
  小升初英语教学课件... — PPT 教学课件
  小升初英语暑期...     — 教学计划
  start.bat / start.sh — 一键启动脚本
  server.js            — 本地服务器（无需安装依赖）


使用方法：

【方式一】本地直接打开（推荐电脑使用）
  双击 index.html 即可在浏览器打开，所有功能可用。

【方式二】局域网访问（平板/手机使用）
  1. 在电脑上双击 start.bat（Windows）或 start.sh（Mac）
  2. 命令行会显示两个地址：
       本机打开: http://localhost:8080
       平板访问: http://192.168.x.x:8080
  3. 在平板的浏览器输入 "平板访问" 显示的地址
  4. 按 Ctrl+C 停止服务


注意事项：
  - 方式二需要电脑和平板连同一个 WiFi
  - Windows 双击 start.bat 如果没反应，可能需要安装 Python
    或 Node.js，或者直接双击 index.html 使用
  - 语音朗读功能需要网络（浏览器在线语音合成）
  - 推荐使用 Chrome / Edge / Safari 浏览器


技术说明：
  本工具为纯前端页面，无后端依赖。
  SpeechSynthesis 语音合成由浏览器内置提供。
  所有数据在页面内，不向任何服务器发送数据。
  离线可用（仅语音功能需要网络）。
